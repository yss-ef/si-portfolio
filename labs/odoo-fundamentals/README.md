# Odoo 17 fundamentals lab

This repository contains the `gestion_etudiants_manuel` module, developed as
part of an academic curriculum to master Odoo 17 development.

## Objectives

The lab demonstrates core principles of Odoo development:
1. Model creation: Defining business objects and field types.
2. Relationships: Implementing Many2one, One2many, and Many2many relations.
3. Interface design: Customizing tree, form, kanban, and search views.
4. Business logic: Adding Python and SQL constraints.
5. Security: Configuring user groups and access control lists (ACL).

## Contents

- `odoo-module/`: Source code for the student management system.
- `reports/`: Detailed reports (TP1-TP5) explaining implementation choices and
  theoretical concepts.

## Key topics

- TP1-TP2: Installation, configuration, and basic models.
- TP3: Constraints and validations (SQL vs Python).
- TP4: Advanced views (Kanban, Graph, and Search).
- TP5: Security and access rights.
