# Odoo 17 Fundamentals Lab

This repository contains the `gestion_etudiants_manuel` module, developed as part of a structured academic learning path to master Odoo 17.

## Objectives
The lab covers the core principles of Odoo development:
1.  Model Creation: Defining business objects and different field types.
2.  Relationships: Implementing Many2one, One2many, and Many2many relations.
3.  Interface Design: Customizing tree, form, kanban, and search views.
4.  Business Logic: Adding Python constraints and SQL constraints.
5.  Security: Configuring user groups and access control lists (ACL).

## Contents
*   `odoo-module/`: The source code for the student management system.
*   `reports/`: Detailed reports (TP1-TP5) explaining the implementation choices and theoretical concepts.

## Key Topics
*   TP1-TP2: Installation, Configuration, and Basic Models.
*   TP3: Constraints and Validations (SQL vs Python).
*   TP4: Advanced Views (Kanban, Graph, and Search).
*   TP5: Security and Access Rights.
